# jqcm-uni
**剑气除魔客户端的uni-app版本**

# 从git上下载代码后使用说明
1. components目录下的以 uni- 开头的组件，是从uni-app插件市场上下载的，所以就没有同步到git上。目前里面有使用的是uni-ui这一套，是创建uni-app采用选择uni-ui项目自带下载的。
